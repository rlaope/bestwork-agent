First-time user guide for bestwork-agent:

1. **What it does**: Overview of bestwork-agent — specialist agents, quality gates, hooks
2. **Available commands**: List all skills and what each one does
3. **Install hooks**: Run `bestwork install` to activate session hooks
4. **First steps**:
   - Try `/bestwork-agent:agents` to see all 38+ specialist profiles
   - Try `/bestwork-agent:trio` to run a task through Tech + PM + Critic gates
   - Try `/bestwork-agent:review` to scan recent code for platform mismatches
5. **Tips**: When to use trio vs. a single agent, how strict mode affects output

Run this whenever you onboard a new team member or set up bestwork-agent in a new project.
